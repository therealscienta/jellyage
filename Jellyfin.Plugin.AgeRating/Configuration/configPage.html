Age Rating Converter

Day-to-day rating fixes live under Dashboard → Age Ratings. This page configures the automation rules.

Converted values are written to each item's Custom rating, not its main parental rating. This keeps the provider's original rating intact across refreshes. Changes always persist in Jellyfin's database; to also save <customrating> to the NFO file on disk, enable the Nfo saver under Dashboard → Libraries → library → Metadata savers.
